# AI 제공자 설정

Hermes는 AI 모델을 사용하기 위해 제공자(Provider)가 하나 이상 필요합니다. Nous Portal의 무료 플랜으로 바로 시작할 수 있고, 더 좋은 조건이 필요할 때 API 키 방식으로 전환할 수 있습니다.

## Nous Portal — 무료로 시작하기

Nous Research가 직접 운영하는 서비스로, **별도 API 키 없이 OAuth 로그인만으로 시작**할 수 있습니다.

**Free 플랜 ($0/mo)**
- `stepfun/step-3.7-flash:free`, `nvidia/nemotron-3-ultra:free` 등 무료 모델 즉시 사용 가능
- 300개 이상 모델 종량제(Pay-as-you-go) 접근

<Callout type="info">
무료 모델은 속도 제한(Rate limit)이 있습니다. 가벼운 테스트와 기본 사용에는 충분합니다.
</Callout>

**유료 플랜 (Plus $20 / Super $100 / Ultra $200)**
- 월정액 크레딧 + 높은 rate limit
- 웹 검색, 이미지 생성, 음성 합성 등 Tool Gateway 포함

## API 키 방식 — 더 좋은 조건이 필요할 때

Nous Portal 외에 Anthropic, OpenAI 등 제공자에서 직접 API 키를 발급받아 사용할 수도 있습니다. 사용한 만큼만 비용이 청구됩니다.

<Callout type="warning" title="API 키는 유료입니다">
API 키를 발급받으면 사용량에 따라 요금이 청구됩니다. 처음에는 소액($5~10)만 충전해서 테스트해보세요.
</Callout>

<Callout type="warning" title="키는 한 번만 보여집니다">
API 키는 생성 직후에만 전체 내용이 표시됩니다. 반드시 즉시 복사해서 안전한 곳에 저장하세요.
</Callout>

### Anthropic (Claude) API 키

1. [console.anthropic.com](https://console.anthropic.com) 접속
2. 회원가입 또는 로그인
3. 왼쪽 메뉴 **API Keys** 클릭
4. **Create Key** 클릭, 이름 입력 (예: `hermes-test`)
5. 생성된 키 복사 (`sk-ant-`로 시작) → 나중에 Hermes 설정에 사용 (`ANTHROPIC_API_KEY`)
### OpenAI (GPT) API 키

1. [platform.openai.com](https://platform.openai.com) 접속
2. 왼쪽 메뉴 또는 상단 탐색에서 **API Keys** 클릭
3. **Create new secret key** 클릭
4. 키 복사 (`sk-`로 시작) → 나중에 Hermes 설정에 사용 (`OPENAI_API_KEY`)

## API 키 안전하게 관리하기

<Callout type="danger" title="❌ 절대 하면 안 되는 것">
- GitHub, GitLab 등 코드 저장소에 업로드
- Slack, 카카오톡 채팅으로 전송
- 공유 메모 앱에 그대로 저장
</Callout>

**권장 저장 방법:**
- **macOS**: 키체인(Keychain) 또는 iCloud Keychain
- **Windows**: Windows 자격 증명 관리자
- **공통**: 비밀번호 관리 앱 (1Password, Bitwarden 등)
