# AI 제공자 설정

Hermes는 AI 모델을 사용하기 위해 제공자(Provider)가 하나 이상 필요합니다. 구독형 Nous Portal과 개별 API 키 방식 중에서 선택할 수 있습니다.

## Nous Portal (구독형 · 추천)

Nous Research가 직접 운영하는 구독 서비스로, Hermes 공식 권장 방법입니다.

**장점**
- 하나의 구독으로 Claude, GPT, Gemini, DeepSeek 등 300개 이상의 모델 사용 가능
- 별도 API 키 없이 OAuth 로그인만으로 시작
- 웹 검색, 이미지 생성, 음성 합성 등 Tool Gateway 포함

**단점**
- 월 구독료가 있습니다
- 구독 없이는 사용 불가

설치 후 아래 명령으로 로그인합니다:

```
hermes setup --portal
```

또는 `hermes model`에서 **Nous Portal**을 선택해도 됩니다.

구독 신청: [portal.nousresearch.com/manage-subscription](https://portal.nousresearch.com/manage-subscription)

## API 키 방식 (사용량 과금)

Nous Portal 대신 직접 API 키를 발급받아 사용할 수 있습니다. 사용한 만큼만 비용이 청구됩니다.

<Callout type="warning" title="API 키는 유료입니다">
API 키를 발급받으면 사용량에 따라 요금이 청구됩니다. 처음에는 소액($5~10)만 충전해서 테스트해보세요.
</Callout>

<Callout type="warning" title="키는 한 번만 보여집니다">
API 키는 생성 직후에만 전체 내용이 표시됩니다. 반드시 즉시 복사해서 안전한 곳에 저장하세요. 이 주의사항은 아래 두 서비스 모두에 해당합니다.
</Callout>

### Anthropic (Claude) API 키

Claude 모델을 직접 API 키로 사용하려면:

1. [console.anthropic.com](https://console.anthropic.com) 접속
2. 회원가입 또는 로그인
3. 상단 메뉴 **API Keys** 클릭
4. **Create Key** 클릭, 이름 입력 (예: `hermes-test`)
5. 생성된 키 복사 (`sk-ant-`로 시작) → `ANTHROPIC_API_KEY`에 입력

### OpenAI (GPT) API 키

GPT 모델을 사용하고 싶다면:

1. [platform.openai.com](https://platform.openai.com) 접속
2. 로그인 후 오른쪽 상단 프로필 → **API Keys**
3. **Create new secret key** 클릭
4. 키 복사 (`sk-`로 시작) → `OPENAI_API_KEY`에 입력

## API 키 안전하게 관리하기

**절대 하면 안 되는 것:**
- GitHub, GitLab 등 코드 저장소에 업로드
- Slack, 카카오톡 채팅으로 전송
- 메모 앱에 그대로 저장 (공유 메모 앱 특히 주의)

**권장하는 방법:**
- macOS **키체인(Keychain)** 또는 **iCloud Keychain** 사용
- 비밀번호 관리 앱 (1Password, Bitwarden 등)

<WindowsNote>
Windows 사용자는 **Windows 자격 증명 관리자** 또는 1Password, Bitwarden 같은 비밀번호 관리 앱을 사용하세요.
</WindowsNote>
