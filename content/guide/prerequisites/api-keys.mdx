# AI 제공자 설정

Hermes는 AI 모델을 사용하기 위해 제공자(Provider)가 하나 이상 필요합니다. 무료로 시작할 수 있는 방법부터 유료 구독, 직접 API 키 방식까지 선택지가 있습니다.

## 무료로 시작하기 — OpenRouter 무료 모델

비용 없이 Hermes를 먼저 체험하고 싶다면 **OpenRouter**의 무료 모델을 사용할 수 있습니다.

1. [openrouter.ai](https://openrouter.ai)에서 회원가입
2. API Key 발급 (무료, 신용카드 불필요)
3. Hermes에 키 입력:

```
OPENROUTER_API_KEY=sk-or-...
```

무료로 사용 가능한 모델 예시 (`:free` 접미사가 붙은 모델):

| 모델 | 특징 |
|------|------|
| `stepfun/step-3.7-flash:free` | 빠른 응답, 일반 작업에 적합 |
| `nvidia/nemotron-3-ultra:free` | NVIDIA 제공, 추론 능력 우수 |

<Callout type="info">
무료 모델은 속도 제한(Rate limit)이 있을 수 있습니다. 가벼운 테스트와 기본 사용에는 충분합니다.
</Callout>

## Nous Portal (구독형)

Nous Research가 직접 운영하는 구독 서비스입니다.

**장점**
- 하나의 구독으로 Claude, GPT, Gemini, DeepSeek 등 300개 이상의 모델 사용 가능
- 별도 API 키 없이 OAuth 로그인만으로 시작
- 웹 검색, 이미지 생성, 음성 합성 등 Tool Gateway 포함

**단점**
- 월 구독료가 있습니다 (무료 플랜 없음)

## API 키 방식 (사용량 과금)

원하는 제공자에서 직접 API 키를 발급받아 사용합니다. 사용한 만큼만 비용이 청구됩니다.

<Callout type="warning" title="API 키는 유료입니다">
API 키를 발급받으면 사용량에 따라 요금이 청구됩니다. 처음에는 소액($5~10)만 충전해서 테스트해보세요.
</Callout>

<Callout type="warning" title="키는 한 번만 보여집니다">
API 키는 생성 직후에만 전체 내용이 표시됩니다. 반드시 즉시 복사해서 안전한 곳에 저장하세요. 아래 두 서비스 모두에 해당합니다.
</Callout>

### Anthropic (Claude) API 키

1. [console.anthropic.com](https://console.anthropic.com) 접속
2. 회원가입 또는 로그인
3. 상단 메뉴 **API Keys** 클릭
4. **Create Key** 클릭, 이름 입력 (예: `hermes-test`)
5. 생성된 키 복사 (`sk-ant-`로 시작) → `ANTHROPIC_API_KEY`에 입력

### OpenAI (GPT) API 키

1. [platform.openai.com](https://platform.openai.com) 접속
2. 왼쪽 메뉴 또는 상단 탐색에서 **API Keys** 클릭
3. **Create new secret key** 클릭
4. 키 복사 (`sk-`로 시작) → `OPENAI_API_KEY`에 입력

## API 키 안전하게 관리하기

<Callout type="danger" title="❌ 절대 하면 안 되는 것">
- GitHub, GitLab 등 코드 저장소에 업로드
- Slack, 카카오톡 채팅으로 전송
- 공유 메모 앱에 그대로 저장
</Callout>

**권장 저장 방법:**
- **macOS**: 키체인(Keychain) 또는 iCloud Keychain
- **Windows**: Windows 자격 증명 관리자
- **공통**: 비밀번호 관리 앱 (1Password, Bitwarden 등)
