# 설치 전 유의사항

설치 전에 아래 사항을 꼭 확인하세요. 나중에 발생할 수 있는 문제를 미리 예방합니다.

## 유일한 필수 조건: Git

Hermes 설치 스크립트가 필요한 것들(Python, Node.js, ripgrep 등)을 자동으로 설치해줍니다. 사용자가 직접 설치해야 하는 것은 **Git 하나**뿐입니다.

<TerminalBlock title="Git 설치 확인">
```bash
git --version
```
</TerminalBlock>

`git version 2.x.x` 같은 결과가 나오면 됩니다. 명령을 찾을 수 없다면 [git-scm.com](https://git-scm.com)에서 설치하세요.

<WindowsNote>
Windows는 Git 설치가 필요 없습니다. PowerShell에서 별도의 설치 스크립트로 진행하며, 설치 페이지에서 안내합니다.
</WindowsNote>

## macOS 보안 설정 확인

### Gatekeeper (보안 경고 처리)
처음 실행 시 "확인되지 않은 개발자" 경고가 나올 수 있습니다.

<TerminalBlock title="보안 경고 발생 시">
```bash
# 설치 후 실행 시 오류가 나면:
xattr -dr com.apple.quarantine ~/.local/bin/hermes
```
</TerminalBlock>

### macOS 버전 확인

<TerminalBlock title="macOS 버전 확인">
```bash
sw_vers
```
</TerminalBlock>

macOS 13 (Ventura) 이상을 권장합니다.

## 네트워크 환경 확인

회사 네트워크나 VPN을 사용 중이라면 API 호출이 차단될 수 있습니다.

<TerminalBlock title="연결 테스트">
```bash
curl -I https://api.anthropic.com
```
</TerminalBlock>

`HTTP/2 200` 또는 `HTTP/2 404`가 나오면 연결 가능합니다. `curl: (6) Could not resolve host` 라면 DNS 문제입니다.

## 체크리스트

설치를 시작하기 전에 아래를 모두 확인하세요:

- [ ] macOS 13 이상 (또는 Windows 10/11)
- [ ] Git 설치됨 (`git --version`)
- [ ] AI 제공자 준비됨 (Nous Portal 계정, 또는 Anthropic · OpenAI 등 API 키)
- [ ] 인터넷 연결 및 API 서버 접근 가능
- [ ] (선택) Gateway 사용 시 Slack 봇 토큰 준비됨 (`xoxb-`로 시작)

<Callout type="tip">
모두 체크됐다면 다음 단계 curl로 설치하기로 넘어가세요!
</Callout>
