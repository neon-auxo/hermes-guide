# Slack 연결

[사전 준비 — Slack 설정](/guide/prerequisites/slack-setup)에서 준비한 **Token**과 **Member ID**가 필요합니다.

1. 앱 좌측 사이드바에서 **Messaging > Slack** 선택
2. **Paste Slack bot token** 입력 창에 `xoxb-`로 시작하는 토큰 붙여넣기
3. **Paste Slack app token** 입력 창에 `xapp-`로 시작하는 토큰 붙여넣기
4. **Save changes** 버튼 클릭
5. 좌측 **New Session** 선택 후 채팅창에 `gateway start 해줘` 입력
6. 채팅창에 `SLACK_ALLOWED_USERS에 <Member ID>추가해줘` 입력
7. Slack에서 초대한 채널에 멘션을 보내보세요.

<TerminalBlock>
```
@Hermes 안녕, 너는 뭘 할 수 있어?
```
</TerminalBlock>

Hermes가 응답하면 연결 성공입니다.

8. cron job 결과와 cross-platform 메시지를 받기 위해 아래 멘션을 보내세요.

<TerminalBlock>
```
@Hermes /sethome
```
</TerminalBlock>

## 응답이 없을 때 점검 순서

`@Hermes 안녕, 너는 뭘 할 수 있어?`를 보냈는데 반응이 없다면:

1. Socket Mode가 ON인지 확인 (Slack 앱 설정 3단계)
2. `message.channels` / `message.groups` / `message.im` / `app_mention` 이벤트가 모두 추가됐는지 확인
3. `channels:history` / `groups:history` scope가 추가됐는지 확인
4. Reinstall to Agents를 실행했는지 확인 (scope·이벤트 변경 후 필수)
5. `/invite @Hermes`로 봇을 채널에 초대했는지 확인
6. DM에서 응답이 없다면 Chat Tab이 ON인지 확인 (Slack 설정 5단계)
7. `SLACK_ALLOWED_USERS`에 내 Member ID가 들어있는지 확인
