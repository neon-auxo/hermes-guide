# GUI 설치 (Hermes Desktop)

가장 쉽게 시작하는 방법은 **Hermes Desktop 인스톨러**입니다. 클릭 몇 번으로 CLI(터미널)와 Desktop 앱이 한 번에 설치됩니다.

Hermes Desktop은 브라우저 웹앱이 아니라 **macOS/Windows용 네이티브 데스크탑 앱**입니다.

## 방법 1 — 인스톨러로 설치 (권장)

1. [hermes-agent.nousresearch.com](https://hermes-agent.nousresearch.com/) 에서 **Download** 클릭
2. macOS 또는 Windows용 인스톨러 다운로드
3. 인스톨러 실행 — Python, Node.js 등 필요한 의존성 자동 설치

<WindowsNote>
Windows에서 인스톨러 실행 시 보안 경고가 뜨면 "추가 정보 → 실행"을 클릭하세요.
방화벽 허용 여부를 묻는 경우 "액세스 허용"을 선택하세요.
</WindowsNote>

<Callout type="tip">
인스톨러로 설치하면 CLI까지 함께 설치됩니다. 다음 페이지의 **curl로 설치하기**는 건너뛰어도 됩니다.
</Callout>

## 방법 2 — curl로 CLI를 설치한 뒤 Desktop 앱 추가

**curl로 설치하기** 페이지에서 CLI만 먼저 설치했다면, 아래 명령으로 Desktop 앱을 추가로 실행할 수 있습니다.

<TerminalBlock title="Desktop 앱 실행">
```bash
hermes desktop
```
</TerminalBlock>

처음 실행 시 Desktop 앱 구성 요소를 자동으로 설치합니다.

## 설치 완료

Hermes Desktop까지 설치했다면 모든 준비가 끝났습니다. 터미널 CLI와 Desktop 앱 중 편한 방식으로 사용하세요.

<Callout type="tip">
Slack, Telegram 등 메신저에서 사용하고 싶다면 **초기 설정 & Slack 연결** 페이지를 참고하세요.
</Callout>
