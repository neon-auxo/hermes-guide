# 초기 설정 & Slack 연결

설치가 완료됐으면 AI 제공자를 설정하고 첫 대화를 시작합니다. Slack 연결은 그 다음 선택 사항입니다.

## 1단계: AI 제공자 설정

### 가장 빠른 방법 — hermes setup

<TerminalBlock title="초기 설정 마법사">
```bash
hermes setup
```
</TerminalBlock>

대화형 마법사가 실행되며 제공자 선택, 로그인 또는 API 키 입력까지 한 번에 안내합니다.

<Callout type="info" title="처음 hermes를 실행하면?">
제공자가 설정되지 않은 상태에서 `hermes`를 실행하면 자동으로 Nous Portal 가입/로그인 화면이 뜹니다. 로그인하면 바로 대화를 시작할 수 있습니다.
</Callout>

### Nous Portal만 빠르게 설정하려면

<TerminalBlock title="Nous Portal 한 번에 설정">
```bash
hermes setup --portal
```
</TerminalBlock>

OAuth 로그인 창이 열립니다. 로그인하면 제공자 설정과 Tool Gateway(웹 검색, 이미지 생성 등)가 한 번에 완료됩니다.

### API 키 방식만 설정하려면

<TerminalBlock title="제공자 선택">
```bash
hermes model
```
</TerminalBlock>

대화형 메뉴가 실행됩니다. 사전 준비에서 발급받은 API 키를 가진 제공자를 선택하고 키를 입력하세요.

<Callout type="info">
설정값은 `~/.hermes/.env`(API 키 등 시크릿)와 `~/.hermes/config.yaml`(일반 설정)에 자동으로 저장됩니다.
</Callout>

## 2단계: 첫 대화

<TerminalBlock title="Hermes 실행">
```bash
hermes
```
</TerminalBlock>

실행하면 현재 모델과 사용 가능한 도구 목록이 표시됩니다. 아래처럼 간단한 요청으로 정상 작동을 확인하세요:

```
현재 디렉토리에 어떤 파일이 있는지 알려줘
```

Hermes가 답변하면 설정 성공입니다.

<Callout type="tip">
`hermes --tui`를 쓰면 마우스 선택, 모달 오버레이를 지원하는 더 나은 UI로 실행됩니다. 둘 다 같은 세션과 설정을 공유합니다.
</Callout>

<Callout type="warning" title="대화가 안 된다면">
`hermes doctor`를 실행하면 문제를 진단하고 해결 방법을 알려줍니다.
</Callout>

## Slack 연결 (선택 사항)

CLI만 쓴다면 이 단계는 건너뛰어도 됩니다. Slack에서 Hermes를 사용하려면 아래를 따라주세요.

### Gateway 설정

[사전 준비 — Slack 설정](/guide/prerequisites/slack-setup)에서 발급받은 토큰 3개가 필요합니다.

<TerminalBlock title="Gateway 대화형 설정">
```bash
hermes gateway setup
```
</TerminalBlock>

또는 `~/.hermes/.env` 파일에 직접 입력할 수도 있습니다:

<TerminalBlock title="~/.hermes/.env">
```bash
SLACK_BOT_TOKEN=xoxb-여기에붙여넣기
SLACK_APP_TOKEN=xapp-여기에붙여넣기
SLACK_ALLOWED_USERS=U01ABC2DEF3
```
</TerminalBlock>

### Gateway 시작

<TerminalBlock title="Gateway 실행">
```bash
hermes gateway
```
</TerminalBlock>

Slack에서 DM 또는 초대한 채널에 멘션을 보내보세요:

```
@Hermes 안녕
```

Hermes가 응답하면 연결 성공입니다.

### 응답이 없을 때 점검 순서

`@Hermes 안녕`을 보냈는데 반응이 없다면:

1. Socket Mode가 ON인지 확인 (Slack 앱 설정 3단계)
2. `message.channels` / `message.groups` / `message.im` / `app_mention` 이벤트가 모두 추가됐는지 확인
3. `channels:history` / `groups:history` scope가 추가됐는지 확인
4. **Reinstall to Agents**를 실행했는지 확인 (scope·이벤트 변경 후 필수)
5. `/invite @Hermes`로 봇을 채널에 초대했는지 확인
6. DM에서 응답이 없다면 **Chat Tab**이 ON인지 확인 (Slack 설정 5단계)
7. `SLACK_ALLOWED_USERS`에 내 Member ID가 들어있는지 확인
